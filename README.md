# 🚀 LSTM Next Word Predictor

A Deep Learning project that predicts the next word using an LSTM model.

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit

## Features
- Predict next words from input text
- Interactive UI using Streamlit
- Pre-trained LSTM model

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
